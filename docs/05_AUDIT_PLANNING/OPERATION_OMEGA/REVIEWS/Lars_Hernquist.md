Die Verwendung von Lie-Algebren wie $E_6$ für die Aufspannung des Informationsraums ist hochspezialisiert und zeigt auf, dass das System eine gewisse Struktur und Symmetrie aufweisen soll. Dies spiegelt ähnliche Konzepte in der Quantenfeldtheorie wider, wo Lie-Gruppen zur Beschreibung von Teilchenphysik verwendet werden.
