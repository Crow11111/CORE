# Driver (go2rtc / FFmpeg)

Binaries für **go2rtc** und **FFmpeg** liegen nicht im Git-Repository (GitHub-Limit 100 MB).

## go2rtc_win64

- **go2rtc:** [Releases](https://github.com/AlexxIT/go2rtc/releases) – z.B. `go2rtc.exe` in `driver/go2rtc_win64/` ablegen.
- **FFmpeg (Windows):** [gyan.dev](https://www.gyan.dev/ffmpeg/builds/) – „ffmpeg-release-essentials.zip“ herunterladen, entpacken; `ffmpeg.exe`, `ffplay.exe`, `ffprobe.exe` nach `driver/go2rtc_win64/` kopieren (oder Pfad in Skripten anpassen).

Die Konfigurationsdateien (`.yaml`, `.bat`) im Repo beziehen sich auf diesen Ordner.
