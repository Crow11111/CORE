# Expertise: UX/UI-Design

## Technologie-Stack

| Stack | Einsatz |
|-------|---------|
| React/Vite | Dev-Agent Dashboard, Admin-Tools |
| Streamlit | Prototypen, interne Tools |

## ND-Optimierung

- **Kognitive Last minimieren**: Weniger Optionen pro Screen, klare Hierarchie
- **Keine überflüssigen Animationen**: Reduziert Reizüberflutung
- **Accessibility**: Kontrast, Fokus-States, Screenreader
- **Vorhersehbarkeit**: Konsistente Patterns, keine Überraschungen

## ATLAS-spezifisch

- Dev-Agent Dashboard: Übersichtliche Session-/Event-Ansicht
- Admin-Tools: Fokus auf Funktionalität, nicht Dekoration
